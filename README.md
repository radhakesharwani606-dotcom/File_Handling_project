# 📁 File Handling System

A Python application for managing files through both a Command-Line Interface (CLI) and a Streamlit web interface. This project demonstrates core file handling operations and Python programming concepts.

## ✨ Features

* Create Files
* Read File Contents
* Update Files
* Rename Files
* Append Data
* Delete Files
* Browse Files and Folders

## 🛠️ Built With

* Python
* Streamlit
* Pathlib
* OS Module
* Git & GitHub

## 🚀 Getting Started

### Run the CLI Version

```bash
python main.py
```

### Run the Streamlit Version

```bash
streamlit run streamlit_app.py
```

## 📚 Key Learnings

* File Handling Operations
* Exception Handling
* Directory Management
* Python Modules
* Building Interactive Apps with Streamlit
